# Muertos y Heridos

## Descripción del juego

El programa piensa un número aleatorio de 4 cifras.

El usuario introduce números hasta acertarlo.

Por cada intento, el programa devuelve una cadena con:

M → número correcto en posición correcta (muerto)

H → número correcto en posición incorrecta (herido)

V → número que no está en el número pensado (vivo)

Ejemplo de salida: MMHV

## Reglas de entrada

- El usuario debe introducir exactamente 4 dígitos.

- Solo se aceptan números.

- Si la entrada no es válida, se vuelve a pedir.

## Objetivo

Adivinar el número pensado por el programa en el menor número de intentos posible.
Al acertar, se muestra un mensaje final con el número de intentos realizados.

## Extras propuestos (para ampliar el ejercicio)

Permitir elegir el número de cifras del número secreto.

Limitar el número máximo de intentos.

Crear un ranking según número de intentos.
